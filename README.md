# 🎓 Transfer Exam RPG Bot (轉學考 RPG 養成計畫)

> **把「備考」變成一場熱血的 RPG 冒險！** > 你的每一分鐘努力、寫的每一份考卷，都將化為角色的戰鬥力！目標是在考試來臨前，達到傳說中的 **Lv.99** 巔峰！🚀

---

## ⚔️ 冒險指南 (Features)

本機器人將枯燥的讀書過程遊戲化，包含以下核心系統：

### 1. 角色系統 (Character)
* **註冊與檔案**：建立專屬考生檔案，記錄等級、屬性與職業。
* **指令**：
    * `/rpg註冊`：初始化角色（身分：🥚 初心考生）。
    * `/rpg狀態`：查看等級、經驗條、三圍屬性及今日修練進度。

### 2. 練功機制 (Leveling Up)
想要變強？必須透過以下方式累積經驗值 (EXP)：

| 方式 | 指令 / 動作 | 獎勵 EXP | 說明 |
| :--- | :--- | :--- | :--- |
| **每日簽到** | `/簽到` | **50+** | 基礎 50 EXP，連續簽到天數越高，獎勵越多。 |
| **每日挑戰** | 自動發送 (00:55) | **150** | 每天凌晨自動派發微積分/物理題目，點擊按鈕解題即可領取。 |
| **讀書回報** | `/讀書 hours:時數` | **20 / hr** | 誠實回報讀書時間（每日上限 10 小時）。 |
| **刷歷屆試題** | `/題目` | **50 / 份** | 查詢並寫完一份考卷後，點擊「📝 我寫完這份了」領取獎勵。 |

---

## 🏰 職業系統 (Class System)

當等級達到 **Lv.5** 時，可使用 `/rpg轉職` 選擇修練道路。不同職業擁有不同的屬性成長與必殺技。

| 職業 | 定位 | 核心屬性 | 成長特色 |
| :--- | :--- | :--- | :--- |
| **📐 微積分大師** | **玻璃大砲** | **智力 (INT)** | 捨棄防禦，追求極致的魔法爆發。升級大幅提升 INT。 |
| **🍎 物理大師** | **重裝戰士** | **力量 (STR)** | 攻守兼備，血量與物理攻擊較高。升級大幅提升 STR。 |
| **💻 計概大師** | **全能型** | **均衡 (All)** | 數值平均，沒有短板，生存能力強。升級平均提升三圍。 |
| **📖 英文大師** | **爆擊流** | **運氣 (LUK)** | 依賴運氣與爆擊，攻擊浮動大但常有奇蹟。升級大幅提升 LUK。 |

> **Note**: 每次升級會自動獲得屬性點，並依據職業特性自動分配。

## 🥊 PVP 決鬥系統 (Duel)

讀書讀到壓力大？找人打一架吧！輸入 `/決鬥 opponent:@對手` 即可發起挑戰。

* **血量 (HP)**：由等級與體力 (VIT) 決定。
* **傷害計算**：物理招式看 **STR**，魔法招式看 **INT**。
* **爆擊機制**：**LUK** 越高，越容易打出 1.5 倍爆擊傷害。
* **技能解鎖**：隨著等級提升 (Lv.1, 5, 10, 20)，將解鎖職業專屬招式（如：羅必達法則、萬有引力墜落、藍屏死機...）。

## 📊 戰情室 (Live Dashboard)

機器人會在指定的 **戰情室頻道** 設置一個 **每 30 分鐘自動更新** 的排行榜。
* 即時顯示全伺服器考生的等級、職業排行。
* 公開監控今日任務狀態（簽到、每日一題），督促大家共同進步。
